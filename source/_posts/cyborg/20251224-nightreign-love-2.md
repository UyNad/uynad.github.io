团队精神不是为了团队，而是为了团队里每个人都能赢。

<!--more-->

---

在个人游戏水平和理解达到一定程度后，单通深夜模式其实比野排要简单得多。不断打磨自己的能力永远是简单的，困难的是和团队其他人进行互补达成共赢。*如果是达成互补，而不是互相拖后腿的话。*

人付出额外努力参与团队合作，是为了取得更宝贵的胜利。对于黑夜君临这款游戏来说，就是更有价值的对局体验。十局比赛里但凡有一两把能遇到共鸣互助的团队，这一把的体验就足够驱使共斗玩家继续玩下去。

虽然通常情况是，十局比赛里有五局公式化发育平平无奇，三局比赛对队友颇有微词，剩下一局濒临散伙崩盘。真正享受或者说能接受团队合作游戏的玩家，不能只打顺风团队体验，也要努力拯救水平不足或氛围不佳的团队，保证能赢得胜利。

当一个拥有团队精神的合作者。这份精神不是为了团队，而是为了自己能赢，以及团队里的每个人都能赢。

<br/>

# 0.充分理解队友的定位和需求

团队合作的开始，理解你的队友。对黑夜君临来说，这一步不需要任何沟通就能完成，你只需要在进入比赛后打开队友遗物，研究他的配装思路。队友的遗物就像求职简历，介绍了自己能做什么和想做什么。*简历可以造假，但遗物不行。*

部分队友的遗物对团队胜利的贡献并不大，例如冷门小众玩法流派，或者游戏理解不足硬堆垃圾词条。但比赛毕竟刚开始，即便是看似水平不足的队友，也能靠局内合作赢得胜利。

<br/>

# 1.第一次简单指令进行破冰

排位模式到达高分段后，绝大部分玩家都知道落地要做什么，无非是快速升到2级后前往三级资源点城寨/教堂。即便如此，也需要在没人标点的时候给出第一个标点。这个标点从沟通结果来说是无意义的，但对沟通氛围来说是有价值的。愿意进行标点意味着“尝试沟通路线”的态度，在不了解对方水平和态度的情况下，通过第一次简单合作尝试互相理解和帮助。这样后续继续进行标点，队友愿意跟随的可能性就会越大。

<br/>

# 2.小事上尽量给予正反馈

在你的能力水平上升后，会学到越来越多的操作技巧和知识，例如宝箱和钥匙点位、杀小怪攒大招、商店的高价值物品和部分怪物的隐藏对策等。小事做得越多，就越有助于团队提高清图效率。但不要把这些小事看做是理所当然的，这会导致你对队友越来越苛刻。尽量对队友做的各种小事给出正反馈，例如标点和表情动作互动。如果队友没做到这些小事，甚至连本身的定位职责都没达成，深究他为何如此是没有意义的。要么自己去做来弥补效率，要么重新调整规划来弥补缺失。

<br/>

# 3.大事上尊重队友但保持底线

一局比赛里的大事，是指会影响整局规划甚至有可能导致崩盘的路线决策。基于0和1的前提，此时团队应该一直保持着标点沟通，且考虑队友的发育需求。如果有队友标记了完全不一样的规划路线，只要评估下来收益大差不差，就尽量尊重队友的思路，从而让他更愿意出力。但如果收益不明确，甚至有崩盘的风险，就重复自己的决策，并且直接开始执行，反逼两个队友跟进上来。

<br/>

# 4.少数服从多数保证团队一致

在3的case里，如果发现反逼失败，两个队友并没跟进，反而是去执行那个你认为不合理的决策的话，唯一的解法只有少数服从多数。无论这个决策多么离谱，掉头回去跟进，并尽量执行出好结果。团队水平低还可能靠运营获胜，团队离心根本不可能赢。离心意味着决策失败的可能性更大，且后续团队很难再有互助氛围。

<br/>

# 5.观察队友水平并针对性养成

在小事和大事得处理上，很容易观察出队友的实际水平。为了保证赢，将资源优先堆给定位更重要且水平更高的队友是正确的。如果自己的角色定位并不关键*（例如神秘跳劈前排）*，在保证自己能完成职责的情况下，一切装备选择都优先投给重要队友。养成队友是为了团队胜利，其次才是满足自己的弥赛亚情结，*如果有这种情结需求的话。*

<br/>

# 6.“管好你自己”

1-5点讲的是如何与队友合作，最基本还是做好自己的职责。包括但不限于，配合团队调整自身的角色定位，做好自己角色应做的分工，最后才是在有余力的情况下尽量配合和帮助队友。如果本局比赛频繁发挥失常，不要想着破罐子破摔，应尽量弥补过错完成职责；不要再想着自己发育起来东山再起，尽量将资源让给更正常人的队友。赢了才是一切。

<br/>

# 7.赛后清算与尝试收编队友

正如开头所说，哪怕是排位模式顶分对局，也并不会把把匹配到值得信赖的队友。有的时候是发挥失常，有的时候是单纯不对付。但无论如何，先赢了再说。赢得比赛后，之前捏着鼻子交互的队友不要心软直接拉黑，觉得有实力的队友进行表情互动尝试刷好感度。如果之后再多遇到几次，可以加好友尝试一起组队游玩。尘世邂逅当然是更有特别体验的，但如果你已经开始期待遇到某个人一起玩，那为什么不主动出击加好友？

<br/>
